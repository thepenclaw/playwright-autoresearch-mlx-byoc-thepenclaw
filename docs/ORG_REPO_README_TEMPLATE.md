# Playwright BYOC Repo README Template

Use this template when creating a new Playwright autoresearch repository.

## Required Sections

1. Scope (Playwright-only)
2. Current experiment table
3. Quick start (`python3 scraper.py`)
4. 10-cycle workflow links
5. Repository layout
6. License

## Suggested Experiment Table Row

| Date | Experiment | Status | Summary |
|------|------------|--------|---------|
| YYYY-MM-DD | `[Experiment Name](experiments/YYYY-MM-DD-name/)` | In Progress/Complete | One-line result |
